# 🎮 MimisTweaks

A powerful and comprehensive client-side utility mod for Minecraft 1.20.4+ that enhances your gameplay with a variety of convenient tweaks and features.

## ✨ Features

MimisTweaks provides a collection of useful utilities designed to improve your Minecraft experience:

### 🎣 **Auto Fishing**
Automatically manages your fishing rod with customizable timings. Set the bite detection threshold and reel-in delays to match your playstyle perfectly.

### 🖱️ **Auto Click**
Automated clicking feature with adjustable delays and click rates. Perfect for repetitive tasks and mining operations. Supports both left-click and right-click actions.

### 📷 **Freecam**
Detach your camera from your player and explore the world freely. Navigate through blocks to scout ahead and plan your adventures.

### 🔍 **Zoom**
Smooth camera zoom functionality with customizable zoom levels. Use the scroll wheel to adjust zoom percentage on the fly.

### 💡 **Fullbright**
Illuminate dark areas without using torches. Adjustable brightness control for maximum visibility in caves and dark environments.

### 🔎 **X-Ray**
See valuable ores and chests through blocks. Customizable ore list lets you focus on finding the resources you need most.

### 🎯 **Target Info**
Display detailed information about targeted entities, including health, equipment, and player stats. Fully customizable display with theme options.

### 📊 **Inventory Sort**
Automatically organize your inventory with alphabetical or custom sorting options. Keep your items organized effortlessly.

### 🖱️ **Mouse Tweaks**
Additional mouse functionality improvements for smoother gameplay interactions.

## ⚙️ Configuration

MimisTweaks features an in-game configuration menu where you can:
- Enable/disable individual features
- Customize keybinds to your preference
- Adjust timings and thresholds
- Change display settings and themes
- Manage X-Ray ore lists
- Fine-tune every aspect of the mod

Configuration is saved to `config/mimistweaks.json` for easy backup and sharing.

## 📋 Requirements

- **Minecraft Version**: 1.20.4 (Java Edition)
- **Modloader**: NeoForge
- **Installation**: Simply drop the JAR file into your mods folder

## 🚀 Installation

1. Download the latest release from CurseForge
2. Install NeoForge for Minecraft 1.20.4
3. Place the MimisTweaks JAR file in your `mods` folder
4. Launch Minecraft with the NeoForge profile
5. Enjoy the tweaks!

## 🔧 Development

This project is built using:
- **Modloader**: NeoForge
- **Build System**: Gradle
- **Minecraft Version**: 1.20.4

### For Developers

If you're interested in modifying or building from source:

1. Clone the repository
2. Open the project in IntelliJ IDEA or Eclipse
3. Run `gradlew build` to compile the mod

If you encounter missing libraries:
```bash
gradlew --refresh-dependencies
```

For a complete reset:
```bash
gradlew clean
```
